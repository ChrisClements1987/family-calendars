📌 Instructions

Google Calendar

  1. Right-click a link above and copy it.
  2. Open Google Calendar.
  3. In the sidebar, click “Other calendars” → “From URL”.
  4. Paste the link and press “Add”.

Outlook Desktop

  1. Right-click a link above and copy it.
  2. Open Outlook → File → Account Settings → Internet Calendars → New.
  3. Paste the link and click “Add”.

iPhone Calendar

  1. Long-press a link above and copy.
  2. Go to Settings → Calendar → Accounts → Add Account → Other → Add Subscribed Calendar.
  3. Paste the link.
